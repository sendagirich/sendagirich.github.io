Katakana!

English spelling

  • In Japanese kana, how we write a character e.g. な is how we pronounce it: / næ /.
  • This is different in English and the Roman alphabet e.g.
    • Same sounds different spelling:
      • no, / nəʊ /
      • know / nəʊ /
    • Same spelling but different sound:
      • live / laɪv / I like live music.
      • live / lɪv / I live in Tokyo.

Katakana English

  • Most English words cannot be written correctly in katakana.
  • Therefore if you write then read everything in katakana you will speak with katakana pronunciation.
  • Yu wiru supi-ku ulaiku zisu.
  • Except for ん, katakana characters end with a vowel. Therefore, loan words (借用語) often have more syllables than in English.

consonant word endings

  • Japanese has only has ん as a consonant word ending. Words usually end with a vowel.
    • katakana
    • loan words
    • Practise consonant word endings so that you do ot add an ア, イ, ウ, エ, orオ to the end of the word.

  • Because katakana loan words usually end in a vowel, even if the English word ends in a consonant, English loan words often have more syllables in Japanese. This is a problem if you don’t change your pronunciation.

Consonant blends

  • Because katakana is always consonant-vowel カ キ ク ケ コ you will add extra vowels (and therefore syllables) when consonants come together (consonants blends) such as pl, st,
  • Consonant blends can be at the begging, middle or end of words.

(2-letter):

  • L-blends: bl-, cl-, fl-, gl-, pl-, sl- (e.g., black, clip, fly)
  • R-blends: br-, cr-, dr-, fr-, gr-, pr-, tr- (e.g., brick, drop, green)
  • S-blends: sc-, sk-, sm-, sn-, sp-, st-, sw- (e.g., sky, smile, stop)
  • Other: tw-, dw- (e.g., twin, dwarf)

(3-letter):

  • spr-, str-, scr-, spl-, squ- (e.g., spring, street, splash)

    Final Consonant Blends

  • -nt, -nd, -nk, -ng (e.g., tent, hand, sink, long)
  • -st, -sk, -sp, -ft, -lt, -lp, -lk (e.g., fast, ask, help)
  • -mp, -pt, -ct (e.g., jump, kept, act)
